# Jeril Kuriakose - Portfolio

Personal portfolio website for Jeril Kuriakose, Principal Data Scientist.

## Live Site

Visit: [https://jerilkuriakose.github.io](https://jerilkuriakose.github.io)

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animations**: Framer Motion
- **Deployment**: GitHub Pages

## Features

- Responsive design with dark/light mode
- Smooth blur-fade animations
- Interactive social dock
- SEO optimized
- Static export for GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This site automatically deploys to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.

**Important**: Make sure GitHub Pages is configured to deploy from "GitHub Actions" (not from a branch).

1. Go to repository Settings > Pages
2. Under "Build and deployment", set Source to "GitHub Actions"
